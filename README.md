# Projeto_Integrado
Cartilha de Segurança Digital para Pequenas Empresas

Mercado Bom Preço: O mercado é um pequeno negócio que utiliza 
computador para controle de vendas, WhatsApp para pedidos, e para negociar e 
Wi-Fi para clientes. Como muitos pequenos negócios, ela possui poucos 
conhecimentos sobre segurança digital, o que pode gerar riscos de vazamento 
de dados ou ataques virtuais
